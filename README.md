# BizSuite – Centralized Business Intelligence Portal

A WhatsApp-integrated Customer Relationship Management (CRM) system designed to help businesses manage customer interactions, sales, and operations — all from one centralized platform.

## 🚀 Features

- **WhatsApp Integration** – Real-time customer engagement via Meta's WhatsApp Cloud API
- **Client Management** – Centralized client database and interaction history
- **Financial Modules** – Invoices, expenses, credit notes, and estimates
- **Project & Task Management** – Track projects, tasks, and team workflows
- **AI Assistant** – Smart automation for customer support
- **Multi-module Dashboard** – Leads, proposals, tickets, and staff management

## 🛠️ Tech Stack

- **Backend:** CodeIgniter 3 (PHP MVC Framework)
- **Database:** MySQL / MariaDB
- **WhatsApp API:** Meta WhatsApp Cloud API (v21.0) via `netflie/whatsapp-cloud-api`
- **Frontend:** HTML, CSS, JavaScript, Vue.js
